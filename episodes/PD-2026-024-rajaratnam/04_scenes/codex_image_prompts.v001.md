# Codex Image-Generation Brief — Scene Prompts
**Episode:** PD-2026-024-rajaratnam — "The Wiretap That Cracked Wall Street"
**Revision:** v001 · **Scenes:** 31 (one masterpiece prompt per distinct [SHOT:])

This brief generates ONE photoreal cinematic documentary still per scene, walking the script in order. Codex is the image generator (SDXL is **not** used here). All on-screen text is added later in Remotion — never render text inside the image.

---

## GLOBAL HARD RULES (baked into every prompt below — read before generating)
- **Aspect / resolution:** 16:9, long edge **≥ 3840 px**, photoreal cinematic documentary still, ultra-detailed, volumetric light, shallow depth of field where appropriate.
- **Brand palette ONLY:** black / midnight-navy base + electric-blue accent **#1F6BFF** + gold **#E5B53A** + silver. Every prompt names the palette. No other hues.
- **No real-person face or likeness** of anyone (no Rajaratnam, no agents, jurors, judge, executives, the four insiders, Gupta). Use anonymizing devices: figures from behind, silhouettes, cropped at the shoulders, hands only, empty chairs, out-of-focus crowds. Stated in every prompt.
- **No real logos, brand marks, corporate seals, court seals, government seals, or identifiable real landmarks/buildings.** Generic / symbolic only. Stated in every prompt.
- **No readable text, letters, numbers, or signage** rendered in the image. Every prompt states "no legible text/letters/numbers."
- **Symbolic reconstruction only** — these are not authentic records and must never be presented as evidence.
- **One image per prompt** — no candidate pool. Regenerate only a failed shot.

## VISUAL SPINE (keep the set cohesive)
Device of the film: **the recorded voice.** Recurring motifs woven across scenes — an audio waveform that freezes into solid "evidence"; reel-to-reel / recorder; glowing phone lines across a dark city; a trading floor that drains from gold to cold blue; an empty courtroom in flat daylight; surveillance-room blues.
**Color arc by movement:** COLD OPEN grey + electric-blue → THE EDGE gold/warm → THE WHISPER NETWORK warm interiors going clinical → THE WIRE cold blue → THE RECKONING courtroom daylight → CODA dusk to blue.
