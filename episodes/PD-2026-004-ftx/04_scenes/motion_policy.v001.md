# PD-2026-004-ftx — Motion / Asset Policy (owner-directed, 2026-06-18)

Owner directive: **"Runway も使うけど全部はやりすぎ。要所で。"** — Use Runway, but not on every shot.
Reserve paid Runway motion for key moments only. Keeps cost down and avoids an over-animated, AI-ish feel
(see [[feedback_video_natural_style]]: natural over flashy).

## Tiering for every shot (assign in shotlist.v001)
- **Tier R (Runway — key beats only, ~6–10 shots max):** the highest-impact moments where real motion
  earns the cost. Candidates for FTX:
  - Cold-open hook — money/number dissolving into nothing.
  - Act II reveal — the hidden door opening / coins flowing through a glowing `allow_negative` line.
  - Act III the run — a crowd/queue surging, balances draining.
  - The verdict beat — gavel / courthouse, weight of the moment.
- **Tier S (MJ still + local SVD subtle motion, FREE — the default for most shots):** gentle push-in,
  parallax, drifting particles/light. Use for the bulk of the episode.
- **Tier K (MJ still + Ken Burns / graphics):** timelines, balance-sheet/number animations, maps, text.

## Rules
- Default a shot to **Tier S/K**; only promote to **Tier R** if it's a narrative payoff or the hook.
- All people = anonymous silhouettes (R2 / no likeness). Runway shots must not generate a real face.
- Natural, restrained grade; no oversaturated "AI" look. Symbolic reconstruction only (never real footage).
- MJ prompts (to be written once the script is locked) will note the intended tier per shot so we know
  which stills also need a Runway pass.
